## Sobre este proyecto
Bienvenido a mi portafolio. Este repositorio es una bitácora de mi evolución en el desarrollo web. A través de 10 ejercicios prácticos, documento mi aprendizaje pasando desde la creación de una simple página en blanco, hasta el desarrollo de interfaces interactivas capaces de realizar cálculos matemáticos en tiempo real. 

El enfoque principal de este proyecto es escribir código limpio, utilizar semántica correcta y garantizar una buena experiencia de usuario (UX).

## Mi Caja de Herramientas
Para dar vida a este proyecto utilicé las tecnologías base del desarrollo web moderno:
* **HTML5:** Para construir una estructura semántica y accesible.
* **CSS3:** Para maquetar, diseñar y dar vida visual a los elementos (uso de variables, flexbox y selectores avanzados).
* **JavaScript (JS):** Para manipular el DOM y dotar de interactividad a la página web.
* **SweetAlert2:** (Implementado en el último nivel) para reemplazar las alertas nativas por ventanas emergentes modernas y elegantes.

## Recorrido por los 10 Ejercicios

Cada archivo representa un escalón en el aprendizaje. Aquí se explica la utilidad técnica de cada uno:

* **Ejercicio 1 - El inicio de todo (`ejercicio1.html`)** Construcción del esqueleto fundamental de cualquier página web (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`) y el clásico despliegue del primer "Hola Mundo".

* **Ejercicio 2 - Jerarquía y Formato (`ejercicio2.html`)** Implementación de etiquetas de texto para dar estructura mediante encabezados (`<h1>` a `<h6>`) y dar peso visual a las palabras con `<strong>` y `<em>`.

* **Ejercicio 3 - Navegación Web (`ejercicio3.html`)** Creación de hipervínculos absolutos y relativos usando la etiqueta `<a>`, controlando la apertura en nuevas pestañas con `target="_blank"`. Además de la correcta inserción de imágenes.

* **Ejercicio 4 - Estructurando Datos (`ejercicio4.html`)** Uso de listas desordenadas (`<ul>`) para agrupaciones genéricas y listas ordenadas (`<ol>`) para procesos cronológicos.

* **Ejercicio 5 - Bases de Tablas (`ejercicio5.html`)** Introducción a la visualización de datos tabulares creando un horario de clases con etiquetas semánticas para tablas: `<thead>`, `<tbody>`, `<tfoot>`, filas (`<tr>`) y celdas (`<td>`).

* **Ejercicio 6 - Separando el Diseño (`ejercicio6.html`)** Evolución del ejercicio anterior. Aquí limpiamos el código HTML y conectamos una hoja de estilos externa (`.css`) para aplicar colores, bordes y el efecto hover en las filas de la tabla.

* **Ejercicio 7 - El Modelo de Caja (`ejercicio7.html`)** Práctica fundamental de CSS para entender el comportamiento de los elementos en pantalla: diferenciando las cajas que ocupan todo el ancho (Block - `<div>`) frente a las que solo ocupan su contenido (Inline - `<span>`, `<a>`).

* **Ejercicio 8 - Tablas Avanzadas (`ejercicio8.html`)** Maquetación de un Horario Escolar complejo. Se aplicaron atributos avanzados como `colspan` y `rowspan` para fusionar celdas (ej. laboratorios de varias horas) y se añadieron selectores CSS como `:nth-child` para un diseño estilo cebra.

* **Ejercicio 9 - Interfaces de Usuario (`ejercicio9.html`)** Desarrollo del Frontend para un Formulario de Contacto ("DevSolutions"). Incorpora contenedores agrupados, inputs específicos (`email`, `tel`), áreas de texto (`textarea`), placeholders y el atributo `required` para validación nativa.

* **Ejercicio 10 - Conversor de Temperatura (`ejercicio10.html`)** La culminación de los conocimientos donde HTML, CSS y JS trabajan juntos. Captura datos del usuario, valida que sean números reales, aplica una fórmula matemática y devuelve el resultado en una caja bloqueada con el atributo `readonly`, todo en tiempo real.
